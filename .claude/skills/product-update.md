# 產品更新規則 Product Update Rules

## 新增產品步驟
1. 更新對應JSON檔案加入新產品資料
2. 重新生成對應分類頁 category/[type].html
3. 生成新產品獨立頁 product/[slug].html
4. 搵圖並處理成600x600px白底JPG

## 修改產品步驟
1. 直接修改JSON檔案對應產品資料
2. 重新生成相關HTML頁面

## Slug命名規則
- 全部小寫英文
- 空格改底線
- 去除特殊符號（括號、斜線等）
- 例：Hennessy VSOP 700ml → hennessy_vsop_700ml
- 例：Glenfiddich 12年 → glenfiddich_12

## 產品卡片必須顯示
- 產品圖片（600x600px白底）
- 中文名稱（大字）
- 英文名稱（細字灰色）
- 規格：容量 x 數量
- 「代理正貨」標籤（如適用，低調金色小標籤）

## 產品獨立頁必須有
- 麵包屑導航（首頁 > 分類 > 產品名）
- 產品圖片
- 中英文名稱
- 規格、酒精度
- 「代理正貨」標籤（如適用）
- WhatsApp查詢按鈕（https://wa.me/85366687448）
- 「你可能喜歡」隨機16個同分類產品橫向輪播

## 代理正貨判斷
- source 欄位係 "代理正貨" → 顯示金色標籤
- source 欄位係 "進口" → 唔顯示標籤
